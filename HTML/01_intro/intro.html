INTRODUCTION OF HTML



What is HTML?

HTML stand for:- Hyper Text Markup Language


What is html in meta tag?

Meta tags always present inside the <Head> tag and are utilized to portray Page portrayals, Certain Keywords, Author of the Document, viewport settings, determining character sets, and so on. They are used by Web Browsers, Search Engines, and other Web Services to rank the web pages accordingly.

Attributes: The following attributes with their values are described below:

  • name: This attribute is used for indicating the character encoding for the HTML Document.

  • http-equiv: This attribute is used to get the HTTP response message header.

  • content: This attribute is used to specify properties value.

  • charset: This is used for indicating the character encoding for the HTML Document.


What is heading tag?

There are six levels of headings defined by HTML. These 6 heading elements are h1, h2, h3, h4, h5, and h6; with h1 being the highest level and h6 being the least.

  • <h1> is used for the main heading. (Biggest in size)
  • <h2> is used for subheadings, if there are further sections under the subheadings then the <h3>elements are used.
  • <h6>for the small heading (smallest one).
